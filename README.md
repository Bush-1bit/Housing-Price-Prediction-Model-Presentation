# House Price Prediction Presentation

## Description
This project uses the Ames Housing dataset to study what factors affect house sale prices. I used linear regression models in R to compare predictors like neighborhood, overall quality, gross living area, lot area, and garage cars.

## Tools Used
- R
- Quarto
- ggplot2
- tidyverse
- tidymodels
- AmesHousing

## What I Did
I cleaned the data, selected important variables, created visualizations, built three linear regression models, and compared how well each model predicted house sale prices.

## Findings
The strongest predictors were neighborhood, overall quality, and gross living area. Model 1 and Model 2 performed better, with R-squared values around 0.77 and 0.78. The model using lot area and garage cars was weaker, with an R-squared value around 0.44.
